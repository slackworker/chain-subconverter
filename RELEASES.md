# Release Notes

## 3.0 Alpha

> 当前对外内测线，更新时间：2026-05-01

### 发布定位

- 当前对外版本阶段：`3.0 Alpha`
- 当前发布基线：`UI-A`
- 默认应用镜像：`ghcr.io/slackworker/chain-subconverter:alpha-latest`
- 默认访问入口：`/ui/a`

`3.0 Alpha` 是当前对外内测发布线，用于收口第三方设备部署、前端主流程和恢复链路；该阶段仍不是 Beta 或正式版。

### 相对 2.0 的重大更新

以下是对用户最直接、最可感知的变化：

- **Docker Compose 一体部署（前后端集成）**：2.0 更多依赖手工拼接运行方式、部署路径分散；3.0 Alpha 提供统一 Compose 路径，前端与后端可一起启动，第三方设备可直接冷启动。
- **可视化下拉菜单配置**：2.0 用户更容易落入“手填参数 + 反复试错”的流程；3.0 Alpha 核心配置转为可视化选择，交互更直观，配置门槛更低。
- **支持短链分享与复用**：2.0 主要依赖长链接分发，复制与传播成本较高；3.0 Alpha 可生成短链接，便于跨设备分享、记录和复用。
- **支持反向解析（恢复配置）**：2.0 通过已有链接回到可编辑状态的能力有限；3.0 Alpha 支持恢复当前配置，方便继续编辑、回看和复用。

### 当前边界

- A/B/C UI 方案评审尚未结束，当前不把任一方案视为最终 UI。
- Alpha 反馈仍在持续收口，尚未进入 Beta 候选冻结。
- 对外发布、第三方设备最小回归与反馈记录模板见 [docs/testing/alpha-release.md](docs/testing/alpha-release.md)。
