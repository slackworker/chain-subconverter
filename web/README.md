# web

Phase 4 前端工程目录。

当前已初始化 `Vite + React + TypeScript + Tailwind CSS` 的主干公共基线，用于承接共享状态模型、公共组件与后续 A/B/C UI 分支方案。

## 常用命令

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
